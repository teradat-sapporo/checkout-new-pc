### commitとpushは別々だった。

VSCodeからだと、用意されているボタンは基本commitのみで、commitとpushを同時にやるのはその横のプルダウンにあった。
けど本来は一緒にやるものじゃないし、いつもcommitとpushはそれぞれコマンド打ってたので、正しい挙動ではある。

なおpushは別にアイコンあって、そこ押せばpushされた。いや初見じゃわからんよ。
